Java 基本物件：Map、HashMap

發表於 2019-11-23 | 分類於 Java

Quick Start

在 java.util 套件（package）中的 HashMap 類別物件，是常用來存放多組成對資料的物件，每組成對的資料包括 key(索引、鍵值) 及 value(值)

建立 HashMap

把 Map、HashMap 當成箱子，放進物品時需要先指定索引，方便後面拿到箱子的人，可以藉由索引取得箱子裡的物品

取出 HashMap

拿到箱子（map）的人，把手伸進箱子，指定索引後可以拿到對應的物品

輸出結果

# Java # Map # HashMap
